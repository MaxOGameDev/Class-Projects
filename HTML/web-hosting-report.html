Research Questions And Answers:

What is a web server? Why do you need one?

A web server is software/hardware that contains websites and allows for connection over the internet, hence you need one to run a site.

Explain why getting a domain is not enough to put your site up on the web

You can own a domain but you need a web hosting company to actually host the website so it can be accessed online.

What service does a website hosting company provide?

They use their servers to contain and host websites so that they can be accessed over the internet.

Research companies that provide web hosting plans. On average, what is the price range(per month)for hosting a website?

The typical minimum hosting plan is on average around $7.99/month without deals.

What are the most important features for you when reviewing a hosting plan?

Cost, storage, and domain registration
