What Is An Internet Domain Name?

An internet domain name is the name of the website/the URL you must type in to reach the website.

Why Are Domain Names So Important On The Web?

Domain names help organize the internet and allow for websites to actually be reachable by anyone from anywhere (relatively speaking)

Why Do Domain Names Have To Be Unique?

If two domain names were the exact same then there would be no way to find out exactly which website you're trying to go to.

What Does It Mean To Register A Domain Name?

Registering a domain name means identifying an IP Address with a name to use in URLs to reach it.

How Do You Acquire An Already Taken Domain Name?

You must purchase the domain from the person who currently holds the name.
