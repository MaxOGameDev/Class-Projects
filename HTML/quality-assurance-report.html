Research Questions And Answers:

What does Quality Assurance mean for website development?

Web Quality Assurance means making sure that a web application is functional and has the best user experience possible.

Describe the two key principles of Quality Assurance: Fit For Purpose and Right First Time

Fit For Purpose means that the application or website functions well for the intended purpose. Right First Time means eliminating as many mistakes as possible.

How does web standard relate to web quality?

Web standard is ensuring the code passes validation tests and as such works as best as it can.

Describe what Web Acessibility means. What steps can you take to make your pages more accessible?

Web Accessibility is about how easy it is for a website to be used. This can improved through methods such as a compliementary/easy to see color pallete, clearly defined links, and multiple methods of navigation built in.
