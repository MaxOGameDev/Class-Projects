Project plan

Title

The Acropolis

Goal

This site exists as a place for those interested in ancient and classical era history to learn more about it and possibly even converse with others who share those interests.

Target Audience

People interested in learning about ancient/classical history.

Main Features

Sections dedicated to teaching about people, places, events, ETC. in regards to ancient history. Hopefully also providing a way for users to discuss it with each other.

Personality

Very informative, antique, or mysterious to fit the ancient theme.

It should help them get into the right curious mood to learn.

Limiting Factors

Basic design is all I can do for now, and chances are pretty high all I'll be able to manage by the end of the quarter is having some basic info pages on history. Likely won't be able to implement a forum.

Styles guide (Change this content so it matches your own plan, not mine)

Background

Background is Sandy Brown

{background-color: #F4A460}

Default font

Text is Dark Gray, 14 pixels

p { font-size: 14px; color: #333333;}

Header

The header section is 235px tall, positioned along the top of each page with the logo centered in the frame. The background is Beige.

header {
height: 235px;
background-color: #f8dea5;
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 120px tall, positioned just below the header with a background color of Brown and Dark Gray font color.

nav { height: 120px; background-color: #995c33; color: #333333; }

Images

Images will be over Brown background with a Dark Gray border around the container

figure { background-color: #995c33; border: 2px solid #333333; }

img { padding: 1em; }

Links

Links will be Gray. On hover, they will have a Light Yellow background and a Dark Gray border.

a { color: #999 }

a:hover { background-color: #ffffed; border 2px dotted #333333; }
